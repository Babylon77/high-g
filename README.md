# High G Solutions - Digital Thread Consulting Website

A modern, professional website for High G Solutions, a defense consulting company specializing in digital engineering and ballistic systems.

## Overview

High G Solutions leverages deep expertise in weapon systems engineering, high-G environments, and ballistic modeling to accelerate the development of precision fires and cannon-based air defense by integrating advanced digital engineering methodologies across DEVCOM AC and industry partners.

## Technology Stack

- **Framework**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Deployment**: Optimized for Netlify

## Getting Started

### Prerequisites

- Node.js 18.0 or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd high-g-solutions
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Building for Production

To create a production build:

```bash
npm run build
```

This will generate a static export in the `out` directory, optimized for deployment on Netlify.

## Deployment

This website is configured for easy deployment on Netlify:

1. Connect your repository to Netlify
2. Set build command: `npm run build`
3. Set publish directory: `out`
4. Deploy automatically on git push
